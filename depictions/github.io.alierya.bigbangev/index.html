BigBangEv
Tweaks



Introduction:A simple BigBang tweak from EVAN!

介绍：移植 Smartisan OS（锤子系统）Or Pin 的大爆炸分词功能，可进行选择复制、翻译、搜索、分享、指令、监听剪贴板等。

4.1.0更新内容：
1.未选取文字时默认全部选中处理
2.监听剪切板功能不再直接弹出BigBang选文字视图，二是增加一个带动画的按钮给用户选择
3.监听剪切板功能中弹出的BigBang按钮支持长按（可设置长按对应的快捷功能）
4.修复某些App初次启动时弹出BigBang按钮的问题

4.0.1更新内容：
1.内测QQ菜单增加
2.尝试修复因网络问题导致弹窗翻译出现安全模式的问题
3.更新支持自动激活，不用再记录激活码，不怕重刷了
4.增加弹窗翻译引擎出现问题或选中的文字存在非法字符时的错误提示

4.0更新内容：
1.修复菜单排序拖动后崩溃的问题
2.增加从服务器同步URL Scheme的功能
3.设计新的UI
4.增加分享按钮
5.增加指令设置功能（通过URL Scheme完成各种自定义功能）
6.增加长按复制按钮 复制到剪切板 不关闭视图功能
7.增加提示菜单位置选项
8.增加背景效果选项
9.删除功能关闭时各种Switch开关不显示的功能
10.增加菜单顺序设置功能
11.修复iPad上横竖屏显示不正确的问题
12.增加指令按钮及相关功能
13.支持微信朋友圈评论的大爆炸
14.弹窗版翻译 增加谷歌翻译引擎
15.增加获取捷径的按钮
16.在弹窗翻译中增加百度翻译引擎
17.适配最新版QQ菜单
18.调整翻译弹窗为异步处理
19.增加共享捷径指令的链接
20.去除没有使用的依赖applist

兼容：iOS 13.0 - 13.6

官方源：https://repo.yfyf.fun/
