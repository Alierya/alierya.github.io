BigBangEv
Tweaks



Introduction:A simple BigBang tweak from EVAN!

介绍：移植 Smartisan OS（锤子系统）Or Pin 的大爆炸分词功能，可进行选择复制、翻译、搜索、分享、指令、监听剪贴板等。

4.4.0更新内容：
1.增加历史清空功能
2.增加文字识别原图校对功能
3.修复搜索的Web界面，使其支持在网页中播放视频，并支持全屏播放
4.为文字校对功能增加开关，默认为开启
5.菜单显示增加选项-可适配FancySelection
更多新增功能，请更新后自行体验。

补充：如果想要使用Snapper2的文字识别为BigBangEv的文字识别并爆炸，安装源内的ZBigBangEvSupport补丁插件，安装后支持Snapper 2识别中文字符。

兼容：iOS 13.0 - 13.7

官方源：https://repo.yfyf.fun/
