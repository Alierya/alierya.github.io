Introduction:Xiaopeng car shields the jailbreak detection, based on version 2.14.0

介绍：小鹏汽车屏蔽越狱检测，基于2.14.0版本，经测试支持新版2.15.8，因作者没有源地址，所以放到本源，方便使用。

原帖说明：我并没有小鹏汽车，也没有使用这个app的需求，做这个插件是因为前段时间有个网友问如何屏蔽这个app的越狱检测，开始我想直接用flex3来完成，折腾了几天发现怎么修改那几个检测越狱的函数都无效，最后砸壳用Hopper Disassembler分析了一下发现竟然是使用C来写的检测，只是检测了常见的几个越狱后存在的目录是否能读取，难怪了，直接hook后返回0，搞定。

兼容：iOS 12.0 - 13.6
